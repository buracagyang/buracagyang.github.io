1. Logistic regression

在spark.ml中，逻辑回归可以用于通过二项逻辑回归来预测二元结果，或者它可以用于通过使用多项逻辑回归来预测多类结果。 使用family参数在这两个算法之间进行选择，或者保持不设置，Spark将推断出正确的变量。

通过将’family’参数设置为“multinomial”，可以将多项逻辑回归用于二元分类。 它将产生两组系数和两个截距。
